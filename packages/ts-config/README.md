## Installation

#### Install the package as a development dependency:

With `pnpm`:

```sh
pnpm add --save-dev @blazzi/ts-config
```

With `yarn`:

```sh
yarn add --dev @blazzi/ts-config
```

With `npm`:

```sh
npm install --save-dev @blazzi/ts-config
```
