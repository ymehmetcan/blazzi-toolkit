## Installation

#### Install the package as a development dependency:

With `pnpm`:

```sh
pnpm add --save-dev @blazzi/eslint-plugin
```

With `yarn`:

```sh
yarn add --dev @blazzi/eslint-plugin
```

With `npm`:

```sh
npm install --save-dev @blazzi/eslint-plugin
```
